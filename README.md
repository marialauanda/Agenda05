# Agenda05
Calculadora de Consumo de Energia 
Este projeto calcula o consumo mensal de energia de um aparelho.

💻 Linguagem Utilizada

Python 🐍

🧮 Fórmula Utilizada

consumoMensal = (potência × horas por dia × 30) / 1000

📊 Exemplo Utilizado

Aparelho: Ar-condicionado
Potência: 9 W
Uso diário: 2 horas

Resultado: 0,54 kWh por mês
